# API reference

Base URL: `http://127.0.0.1:5000`. Interactive docs (try requests live): **`/docs`** (Swagger)
and **`/redoc`**. The OpenAPI schema is at `/openapi.json`.

All routes are defined in [app/api/routes.py](../app/api/routes.py).

## `GET /api/models`

Lists every selectable model, cloud + local merged.

```json
{
  "models": [
    {"id": "@cf/meta/llama-3.3-70b-instruct-fp8-fast", "kind": "cloud"},
    {"id": "llama3.2:3b", "kind": "local"}
  ],
  "default": "@cf/meta/llama-3.3-70b-instruct-fp8-fast"
}
```

`kind` is `"cloud"` (Cloudflare) or `"local"` (Ollama, discovered live from `/api/tags`).

## `POST /api/chat` ‚Äî streaming completion (SSE)

Request body (`ChatRequest`):

```json
{
  "messages": [{"role": "user", "content": "Hello"}],
  "model": "@cf/meta/llama-3.3-70b-instruct-fp8-fast",  // optional; defaults to server default
  "max_tokens": 2048,            // optional, 1‚Äì32768
  "temperature": 0.7            // optional, 0‚Äì2
}
```

- `role` must be one of `system` | `user` | `assistant` (else `422`).
- `temperature` outside `0‚Äì2`, `max_tokens` outside `1‚Äì32768`, or missing `messages` ‚Üí `422`.

**Response:** `text/event-stream`. Each line is `data: {json}\n\n`, where the object is one of:

| Event | Meaning |
|---|---|
| `{"thinking": "‚Ä¶"}` | Hidden reasoning chunk (reasoning models only) ‚Äî UI shows it collapsed. |
| `{"token": "‚Ä¶"}` | A chunk of the visible answer; concatenate in order. |
| `{"error": "‚Ä¶"}` | A user-friendly error (no text returned / needs credit / interrupted / raw message). |
| `{"done": true}` | Terminal event ‚Äî the stream is complete. |

The service retries once on an empty response and backs off up to 3√ó on transient upstream
errors before emitting `{"error": ‚Ä¶}` + `{"done": true}`.

Example consumed stream:

```
data: {"token": "Hello"}
data: {"token": "!"}
data: {"done": true}
```

## `POST /api/extract` ‚Äî file ‚Üí text

`multipart/form-data` with a `file` field. Extracts text from `.txt`/`.md`/`.pdf`/`.docx`
(see [app/llm/document_loader.py](../app/llm/document_loader.py)), then deletes the temp upload.

```json
{ "text": "‚Ä¶extracted‚Ä¶", "name": "report.pdf", "chars": 1234 }
```

Unsupported types / read errors ‚Üí `400 {"error": "‚Ä¶"}`.

## `POST /api/title` ‚Äî auto chat title

Body `{"messages": [...], "model": "‚Ä¶"?}`. Returns a short 3‚Äì6 word title generated by a
non-streaming completion. Errors ‚Üí `400 {"error": "‚Ä¶"}`.

```json
{ "title": "Bug in retry logic" }
```

## Chat history (SQLite backup)

The browser owns history in localStorage; these endpoints are a best-effort durable backup.

| Method | Path | Body | Response |
|---|---|---|---|
| GET | `/api/chats` | ‚Äî | `{"chats": [ {id,title,messages,created,updated,titled}, ‚Ä¶ ]}` |
| POST | `/api/chats` | array of chat objects | `{"saved": <count>}` (upsert) |
| DELETE | `/api/chats/{chat_id}` | ‚Äî | `{"deleted": "<chat_id>"}` |

A chat object:

```json
{
  "id": "c_abc123",
  "title": "My chat",
  "messages": [{"role": "user", "content": "hi"}],
  "created": 1719500000000,
  "updated": 1719500050000,
  "titled": true
}
```
